# ClubPoster AI — SaaS Pro MVP

Projet Next.js prêt pour Vercel.

## Installation locale
```bash
npm install
npm run dev
```

## Déploiement Vercel
1. Crée un repo GitHub.
2. Mets tous les fichiers du dossier dedans.
3. Sur Vercel : Add New > Project > Import GitHub.
4. Clique sur Deploy.

## Important
Cette version est un MVP : elle génère une affiche et permet de la télécharger en PNG.
Les comptes utilisateurs et Stripe sont prévus dans la prochaine étape.
